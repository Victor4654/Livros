# 📚 Biblioteca Dev — Trilha de Dados

Repositório pessoal de livros técnicos organizados em uma trilha de leitura com foco em **Python** e **Engenharia/Ciência de Dados**.

A ordem das pastas reflete a sequência recomendada de leitura — cada fase constrói sobre a anterior.

---

## 🗺️ Trilha de Leitura

```
Fundamentos em Python → Código de Qualidade → Arquitetura de Software → Área de Dados
                ↑
        Leitura Paralela (qualquer fase)
```
